Bước 1: Thầy download source code về và giải nén source ra
Bước 2: Vào SqlServer, thầy tạo database mới có tên là ExamOnlineDb_HK2_2021, thầy mở file script ScriptExamOnlineEnglish.sql kèm trong source code, và thầy chạy script này để tạo database và các table của nó
Bước 3: Thầy vào source code, tìm đến file appsetting.json, thầy đổi ConnectionStrings theo cấu hình bên máy thầy tương ứng, sau đó thầy build và run project!
